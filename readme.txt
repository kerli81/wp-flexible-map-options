Provides an option page for the plugin WP Flexible Map to define the default values globally.

== Description ==
The plugin [WP Flexible Map](http://wordpress.org/plugins/wp-flexible-map/) provides only shortcode attributes to configure the behaviour of the map. This plugin enhances provides an option page to set the values once for alle maps. It is still possible to override the values by using the attributes in the shortcodes.

== Installation ==
1. Upload this plugin to your /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the provided option page to configure your global parameters
